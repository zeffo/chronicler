CHRONICLER is a fast and flexible solution to the hassle of navigating the SICSR timetable website to find your classes.

Add a RULE for each course. If the course has classes for multiple divisions, you can add FILTERs, which are regular expressions, to exclude any irrelevant classes.
Once you've created your rules, they'll be stored on your device forever.

An example rule:

Type


This will include all classes under the type BCA (IV) - Elective - Group II - GroupA. Since this includes class for both divisions A and B, we can use a regex filter to include only the classes for division A. Div.? ?A will filter out any results that do not contain Div A, Div.A, Div. A or DivA.
